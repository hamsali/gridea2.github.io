文件管理技巧

2021-04-18 经验分享

文件管理技巧

  • 本地文件先测试，多检查，让问题的频率低些。解决当前问题解决，少产生版本。
  • 做好文件的备份，同步和共享。
  • 多人做的时候，会有很多版本分支，需要权衡好，生成版本号。相当于文件传递，检查，归档。
  • 命名统一做好归类，知识体系-分类-文件名-关键词-版本号-作者-日期。
  • 善用搜索，可以是本地检索，也可以是上传后的标签筛选。
  • 定期管理文件，不要保存不需要的文件，简单最好。只整理20%的关键文件，其他的可以不保留或归到容纳箱
  • 同一目录的文件不要太多，容易检索不便。
  • 重点维护经常使用的文件，桌面可以增加快捷键。

环境

参考原文：

添加清华镜像

conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main/
conda config --set show_channel_urls yes

更改目录

改配置文件：

jupyter notebook --generate-config

可以看到jupyter_notebook_config.py文件路径，打开该文件：

查找参数c.NotebookApp.notebook_dir并设置其路径，设置完成后并保存退出，如上图所示。

将
#c.NotebookApp.notebook_dir = ''
改为
c.NotebookApp.notebook_dir = 'your path'
我改成了我自己的路径
c.NotebookApp.notebook_dir = 'E:\PY'
我已经将文件建好并创建了一个txt文件

注意：第一、文件夹必须先创建好，不然会jupyter初始化时会找不到目录
第二、要取消注释，c前面的#要去掉。
第三、要注意文件名不可以是数字

保存并关闭。
现在可以在Anaconda Navigator中打开

希望用快捷方式打开

在jupyter notebook快捷方式的属性里要删掉%USERPROFILE%
%USERPROFILE%会使得jupyter打开固定地址。
